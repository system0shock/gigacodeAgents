# Branch Naming Rules

Recommended branch formats:

- `feature/<task-slug>`
- `bugfix/<task-slug>`
- `fix/<task-slug>`

Examples:

- `feature/payment-retry`
- `bugfix/card-blocking-timeout`
- `fix/null-customer-status`

Используйте lowercase ASCII, digits и hyphens. Не используйте spaces, ticket-only branch names и имена, похожие на protected branches.
